# Resume Website

## Description
This repository contains the source code for my professional resume website, highlighting my skills and experience in information technology and software engineering.

## Technologies Used
tomtapia/tomtapia.github.io is built on the following main stack:

- [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Bootstrap Framework](https://getbootstrap.com/)

Full tech stack [here](/techstack.md)

## Installation
```bash
git clone https://github.com/tomtapia/tomtapia.github.io
pnpm install
pnpm start
```

## Development Security Notes
- The Vite dev server and preview server are intentionally bound to `127.0.0.1`.
- Do not expose Vite to LAN or the public internet unless that change is explicitly required and risk-reviewed first.
- Security hotfixes for Vite are tracked in issue `#18`, with the dependency upgrade handled in PR `#17`.
